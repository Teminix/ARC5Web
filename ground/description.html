About this Website



This website has been created by Team ARC5 which shows the total energy usage in a house through the use of sensors. The website takes in the data given by the sensors, interprets it then displays it. The wesbite provides detailed information about the energy usage it the user's house like time switched on, total usage etc. The user can view the energy consumption of different devices which were running in the designated house. By looking at thier daily, weekly and monthly statistics, the user can analyze electrical usage of differnt types of appliances. The website is still in the development stage and is still in the process of improvement
